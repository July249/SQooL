# SQooL 🏫

웹페이지에서 실행 가능한 교육 목적의 SQL 튜토리얼 사이트

## 목적

본 저장소는 개발자가 SQL을 좀 더 쉽게 접하고 학습할 수 있는 웹페이지를 제작하는 데 목표를 두고 있습니다.
비영리 프로젝트이며, 조건 없이 사용할 수 있습니다.
